## Build

This Doc is develop by docsify.

First, install docsify-cli:

```bash
sudo npm i docsify-cli -g
```

get docs, then start:

```bash
https://github.com/codexnetwork/Documentation.git
cd Documentation/docs
docsify serve .
```

then, visit `http://127.0.0.1:3000`.
