# Codex 开发者文档

---------------------------------------

https://github.com/codexnetwork/Documentation/#/

本文档基于docsify文档系统

安装 docsify-cli:

```bash
sudo npm i docsify-cli -g
```

国内npm较慢，可以使用cnpm

从github上拉取项目，使用docsify可以在本地启动

```bash
https://github.com/codexnetwork/Documentation.git
cd Documentation/docs
docsify serve .
```

访问 http://127.0.0.1:3000.
